# PSA Plan Optimizer
- PSA鑑定プランの期待利益を計算して、1日あたり期待利益が最大のプランを提示するSwiftUIアプリです。購入価格・PSA10価格・PSA9価格・PSA10取得率を入力すると、各プランの比較表と最適プランを表示します。
-------------------------------------------------------

## 主な機能
- PSA10/PSA9の期待販売額を使った期待利益計算
- 手数料・諸費用・鑑定料を反映した比較
- 納期（営業日）から実日数に換算し、予定返却日を表示
- プラン料金/納期、販売手数料、諸費用の設定画面
## 計算の前提
- PSA10価格が空欄の場合は購入価格の2倍
- PSA9価格が空欄の場合は購入価格の1倍
- PSA10取得率のデフォルトは80%
- 販売手数料のデフォルトは10%
- 諸費用のデフォルトは240円
